# Hope for Kids Website

## Project Overview
This project is a simple, responsive website for the non-profit organisation **Hope for Kids**.  
It includes six main pages: Home, About, Services, Gallery, Contact, and Enquiry.

---

## Part 3 – Enhancing Functionality and SEO

### Changes Made

#### JavaScript Enhancements:
- Added interactive lightbox gallery for images
- Implemented form validation for contact and enquiry forms
- Added search functionality for services page
